# Buttons

Buttons communicate an action to happen on user interaction.

---

Rubrik supports four different button styles, in multiple sizes and with an optional `reverse` style (for use on alternate light/dark backgrounds).  


## Types

### Alpha
An Alpha button is always used for the most important actions.  
Only one Alpha button should be used for any action, such as a form submission, or primary link.

<Playground>
    <Button.Alpha size="lg">Alpha button</Button.Alpha>
</Playground>

---
