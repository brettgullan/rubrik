# Typography

Rubrik's type scale is shamelessly lifted from the BBC's Global Experience Language (GEL).  

---

There are 10 steps in the primary type scale, plus an additional 5 sizes used for specialty applications -- display headings, infographics, etc.

The type scale provides a distinct set of sizes, suited to a wide range of typographic applications -- from titles and headings, to body copy, captions, ledes and labels.

Beyond establishing the size scale for text, Rubrik (as per BBC GEL) places no other restrictions on type application. Typefaces, weights, styles and colours can be used as appropriate for the particular application.

In most cases, it will probably *not* be necessary to use all 10 sizes in the type scale in a single application. Consumers should free to pick-and-choose the most appropriate sizes.

Alternatively, the font sizes can be adapted by configuring the theme values.

## Import

```javascript
import { Type } from '@rubrik/core
```

Alternatively, Type Components may be imported individually by name:

```javascript
import { Canon, Trafalgar, ... } from '@rubrik/core
```
