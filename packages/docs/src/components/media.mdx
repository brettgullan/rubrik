<head>Media</head>

<lede>
Responsive media components are a primary focus of Rubrik.  
The media package provides responsive Image, Picture and Video components.
</lede>

The `@rubrik/media` package provides a `<ResponsiveImage>` and `<ResponsivePicture>` component that work together with a `<ResponsiveProvider>`.
