## Usage

The `<Tile>` and `<TileLayer>` components are used together to create layered compositions. 
One or more child `<TileLayer>` components can be used. Each `<TileLayer>` is rendered as absolutely positioned content within an enclosing `<Tile>` component.
Any content **not** contained within a `<TileLayer>` will be rendered as static positioned content, forming the *base* layer.  

By default, both `<Tile>` and `<TileLayer>` components are configured to use `display: flex` allowing content to be easily positioned within the Tile.

<Playground>

    <TileLayer justifyContent="center" alignItems="center">

      </Heading>
    </TileLayer>
  </Tile>
