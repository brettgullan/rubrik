<lede>A convenience component that automatically stretches to fill the available space.</lede>  

## Import

```javascript
import { Stretch } from '@rubrik/core
```

## Usage

Use to declare that a Flex component will stretch to fill all remaining space in a component.
Particularly useful when composing card layouts in a Flex-based grid, where the container component height is indeterminate.

    <Flex flexDirection="column" height="400px" bg={'gray.2'}>
      <Text variant="primer" color={'white.0'} p={4}>
        Top
      </Text>
      <Stretch
        bg={'brand.primary'}
        p={4}
        display="flex"
        alignItems="center"
        justifyContent="center"
      >
        <Text color={'white.0'} variant="paragon">
          This area stretches to fill the available space.
        </Text>
      </Stretch>
      <Text variant="primer" color={'white.0'} p={4}>
        Bottom
      </Text>
    </Flex>
